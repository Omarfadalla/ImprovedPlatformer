
# Improved Platfromer (2ND Project)

A brief description of what this project does and why it mas made and incomplete




## Project Overview

this project is purely made in Godot 4 engine and features a 2d pixel art game which contains a player a level and a drone .

the player can move and is fully animated and consists of 2 parts top and bottom .

the level is created via godot's tilemap layer system and collisions have been added to the level 

A fully bullet control system with crosshair animation are also created 

Drone is fully animated and detects player within an area using area 2d node and body entered and exited signals to ensure the drone doesnt always chase the player.


## Controls

Use WASD for player movement 

Use mouse for aiming

Use left Mouse Click for shooting
